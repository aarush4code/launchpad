
I created a macropad, that can:
- higher and lower the volume
- copy and paste
- enter
- pause.

Whilecreating this, I learned how to wire, route, and CAD things in Fusion360. This project has been very fun as making a macropad in something I have never done before.

## Bill of Materials (BOM)
Here is the list of physical materials for launchpad:

- 1x Seeed Studio XIAO RP2040
- 6x Cherry MX Switches
- 6x DSA Keycaps
- 6x SK6812 MINI-E LEDs
- 4x M3x16mm Screws
- 4x M3x5mm Heatset Inserts
- 1x 3D-printed case (with 2 pieces)
